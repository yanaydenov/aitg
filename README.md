# aitg — Telegram AI userbot

Юзербот на базе Telethon + OpenRouter. Отвечает на `.ai ...` в любом чате, понимает фото/видео/голосовые, читает PDF и документы, делает саммари YouTube-видео, ставит напоминания, генерирует стикеры, запускает Python-код и многое другое.

## Стек

- [telethon](https://docs.telethon.dev/) — userbot-клиент
- [openrouter](https://openrouter.ai/) — LLM API (Gemini, Claude, GPT и др.)
- SQLite — память (per-chat, global, per-user, reminders, conversation log)
- `ffmpeg` (системный) — кадры из видео, конвертация аудио
- `Pillow` — конвертация стикеров в WebP
- `pdfminer.six`, `python-docx` — чтение PDF и DOCX
- `youtube-transcript-api` — транскрипты YouTube
- ddgs, trafilatura, httpx — веб-поиск и парсинг

## Установка

```bash
python3.11 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
apt install ffmpeg   # или brew install ffmpeg

cp .env.example .env
# заполни TG_API_ID, TG_API_HASH, OWNER_ID, OPENROUTER_API_KEY

# первый запуск — телеграм спросит телефон и код
python bot.py
```

## Команды

| Команда | Описание |
|---|---|
| `.ai <вопрос>` | задать вопрос / дать задачу |
| `.ai` реплаем на фото/видео/аудио/документ | анализ медиа |
| `.stop` | приостановить бота (только владелец) |
| `.start` | возобновить работу бота |
| `.add <user_id>` | добавить пользователя в whitelist (в ЛС с владельцем) |
| `.remove <user_id>` | удалить пользователя из whitelist |

## Возможности агента

**Медиа и файлы**
- Анализ фото, видео, GIF (кадры через ffmpeg)
- Понимание голосовых сообщений (mp3 → Gemini audio)
- Чтение PDF, DOCX, TXT, CSV и других документов
- Просмотр профилей пользователей с аватаркой (vision)

**Контент**
- Генерация и редактирование изображений (по фото или описанию)
- Создание стикеров из фото или по описанию (512×512 WebP)
- Саммари YouTube-видео по транскрипту
- Перевод с сохранением сленга, тона и стиля оригинала

**Поиск и чаты**
- Веб-поиск, чтение страниц, погода, курсы валют, крипта
- Чтение истории текущего и других чатов, фильтр по периоду
- Умный поиск каналов по теме (`list_channels`, `search_chats`)
- Поиск сообщений по тексту по всем чатам

**Память и автоматизация**
- Напоминания: `напомни через 30 минут ...` / `напомни завтра в 12 ...`
- Долгосрочная память: per-chat, глобальная, per-user
- Полный лог всех диалогов с поиском
- Запуск Python-кода (только для владельца)
